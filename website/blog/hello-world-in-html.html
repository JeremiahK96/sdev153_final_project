Hello World in HTML

  • Programming
  • Jeremiah Knol
  • 2023.10.13

The basics of HTML is not difficult to learn. Unlike many programming languages, HTML is simply for marking up documents, so it isn't a true "programming language" like C or Java.

Anotomy of a Basic HTML Document

Every HTML document follows the same basic root layout. This consists of a document type declaration, the <html> element, the head, and the body.

The homepage of a website is typically saved in a file called index.html, so save the following code to a text file with that exact name:

<!DOCTYPE html>

<html>
	<head>
	</head>

	<body>
	</body>
</html>

The first line is the document type declaration, and it tells your browser that this is in fact an HTML document.

Next we see the beginning of the html element. It is "opened" with <html>, and then "closed" on the last line with </html>, almost like a "begin parentheses" and "end parentheses". Every element within HTML code should be opened and closed in this way.

Elements can be contained inside of other elements. As you can see, we are opening and closing both the head and body elements while inside of the html element. All of your HTML code must be contained within the html element.

Adding Content

At this point, we have a document "skeleton", but there is nothing in it yet. You can actually open this file in your browser, but it will be a blank page. The next step is to fill in some content for your webpage!

Adding a Title

The first thing we will add is a title for the page, which will be displayed on the tab within the browser. To do this, all we need is to add a title element inside of the head element.

	<head>
		<title>My First Webpage</title>
	</head>

If you refresh your browser now, you will see your page title shown in the tab!

Adding a Heading

The head section of an HTML document is used to store metadata for the document itself, such as the title. But this is not displayed on the page. To add content to the page, you simply add it into the body element.

The first thing you probably want to add is the heading for your page, using the h1 element. It should be the full title of your webpage, and you should only have a single h1 element for each webpage.

	<body>
		<h1>My First Webpage</h1>
		<p>Hello World</p>
	</body>

We also added a p tag, which is the paragraph tag, so we could add some text. You can add as many of these as you want!

Complete Example

Here is the full version of the file we created:

<!DOCTYPE html>

<html>
	<head>
		<title>My First Webpage</title>
	</head>

	<body>
		<h1>My First Webpage</h1>
		<p>Hello World</p>
	</body>
</html>
