How to Compile Your First C Program

  • Programming
  • Jeremiah Knol
  • 2023.09.23

Scripting languages like Python or Bash are interpreted into machine code on-the-fly, which means that the program itself is the source code. But C is a compiled language, so you must compile your source code into machine code first before being able to execute it. Jump start your journey into C by learning how to compile your first C program!

Setup Your Compiler

There are multiple options when it comes to compilers for C, but for this article we will stick with gcc. Make sure you have it downloaded and installed before moving on to the next step, or you will not be able to compile anything!

Write the Program

Now it's time to get started on the code! Here is the example we will be running, save it to the file hello.c:

#include <stdio.h>
int main() {
	printf("Hello World\n");
	return 0;
}

In C, main is the name of the function that must be used for the starting point of your program. It can actually take arguments from the command line, but in this example, we won't use any. We are simply including the standard input/output header called stdio.h, which gives us the printf() function which can output characters to the terminal.

Compile the Program

Now that we have the source code done, we need to compile it to machine code. Run the following command in the terminal, from the same directory as the source code file:

gcc -o hello hello.c

In this command, gcc is the compiler we are running, -o hello means we want to output the machine code executable to a file called hello, and hello.c is the source code we want to compile.

Now, if you execute the hello executible, you will see Hello World printed to the terminal! Congratulations, you just made your first C program!
